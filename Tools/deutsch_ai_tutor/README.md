# Deutsch AI Tutor 🇩🇪

An AI-powered German language learning app built with Flutter and Claude AI.

## Features

- **Daily AI Lessons** — Personalized lessons from A1 to C2, generated by Claude AI
- **Homework & Grading** — Auto-generated homework with AI correction and detailed feedback
- **Audio Practice** — Text-to-speech pronunciation with slow-mode for every word and sentence
- **Progress Tracking** — XP, streaks, level progression (A1→C2), charts, and achievements
- **Resources** — Curated YouTube channels, podcasts, and websites + AI-picked recommendations
- **Daily Reminders** — Push notifications to keep your streak alive

## Setup

1. Get a free API key from [console.anthropic.com](https://console.anthropic.com)
2. Run the app and enter your key on the onboarding screen

```bash
flutter pub get
flutter run
```

## Tech Stack

- Flutter (Dart)
- Claude API (`claude-opus-4-6`)
- `flutter_tts` — German text-to-speech
- `fl_chart` — Progress charts
- `flutter_local_notifications` — Daily reminders
- `shared_preferences` — Local storage
- `flutter_markdown` — Lesson rendering
- `url_launcher` — Open external resources
